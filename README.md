# stlm-webshelf
Webshelf sytem acts as an online clipboard for users to access from anywhere

## The webshelf
The webshelf is a simple one page-app, that allows the user to dynamicly add simple elements like
- links
- code snippets
- text snippets
- contacts
- small file
- credentials
- secrets

The webshelf is ment to help people work efficiently accross multiple devices and plattforms.
It offers a clipboard-of-a-kind feeling for users who may want to use a codesniped or api-key generated on a different device.
This is very usefull when working on someone else's computer, or whilst setting up a new machine.
Imagine you setup a brand new system and want to run your installation script -> how do you get the script to the new machine?
- do you log in to some cloud storage
- do you carry your script on an usb-stik
- do you send it via email and log into your webmail

Why not just quickly open your webshelf and copy it from there?

## Shelfsessions
if you want to access your webshelf, you can obviously log in via credentials. But this brings the drawback of
having the credentials ready (either know them by heart or type them up).
This is where Shelfsessions come in handy.
1. If you are unauthorized you can create a new session
2. With your mobile device (on which you are obviousely logged in at any time) you scan the generated session-qr-code
3. The session is now granted access to you shelf

## Shelf components
As a user you can create various components to shelf some important data.

### Links
You can add simple web-links in a list, to quickly access your favourite sites

### Code-Snippets
You can add small unformatted snippets. The provided 'Copy to Clipboard' Button allows for simple access

### Text-snippets
You can add text-snippets containing formatted text.

### Contacts
Contacts with email and phone-number.

### Files
Important files.
